## Getting Started (Local Development)

To run this project locally or compile the Tailwind CSS file, you can now use Node.js and NPM:

1. **Install dependencies**:
   ```bash
   npm install
   ```
2. **Watch for Tailwind CSS changes**:
   ```bash
   npm run watch:css
   ```
   *Note: This will watch the `src/tailwind-input.css` file and build it into `tailwind-built.css` in the root directory.*

3. **Build CSS once**:
   ```bash
   npm run build:css
   ```

*(The old `server.ps1` script has been deprecated in favor of this cleaner package management approach.)*
