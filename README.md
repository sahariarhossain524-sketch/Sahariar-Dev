## 🚀 Featured Projects Included

1. **GrowthEdge Consulting**: A modern, responsive landing page for a strategic consulting firm. Features clean typography and smooth reveal animations.
2. **Alex Carter Portfolio**: A sleek, personal portfolio template designed for creative professionals with modern CSS layouts.
3. **UrbanNest Real Estate**: A highly visual property listing landing page focusing on structured component design and mobile-first responsive interfaces.

## 💻 Tech Stack

| Technology | Description |
|------------|-------------|
| **HTML5** | Semantic markup for clean structure |
| **Tailwind CSS** | Utility-first styling for rapid, responsive design |
| **CSS3** | Advanced animations, flexbox, and grid layouts |
| **Vanilla JS** | Lightweight interactions and DOM manipulation |

## 🛠️ Usage

This repository contains static frontend projects. No complex build tools are required!

1. **Clone the repository**
   ```bash
   git clone https://github.com/sahariarhossain524-sketch/Sahariar-Dev.git
   ```

2. **Open in Browser**
   Simply open any `.html` file (e.g., `growthedge.html`) directly in your web browser, or use a tool like VS Code Live Server for hot reloading.
